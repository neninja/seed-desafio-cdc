# Contribuindo

## Desenvolvimento

### Ambiente

- Baixe JDK 21
- Baixe as dependências
```shell
make clean
```

### Execução

- Inicie a aplicação
```shell
make run
```

> Caso queira utilizar debug, execute `make debug` e conecte o Intellij como *Remote Debug* na porta 5005

### Padronização

- Formate o código
```shell
make fmt
```
